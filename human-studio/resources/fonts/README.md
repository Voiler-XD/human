# Bundled Fonts

Place these font files here before building:

- `Nunito-Bold.woff2` — Logo typeface
- `NunitoSans-Regular.woff2` — Body text
- `NunitoSans-SemiBold.woff2` — UI emphasis
- `NunitoSans-Bold.woff2` — Headings
- `JetBrainsMono-Regular.woff2` — Code editor and terminal

Download from:
- Nunito/Nunito Sans: https://fonts.google.com/specimen/Nunito
- JetBrains Mono: https://www.jetbrains.com/mono/
